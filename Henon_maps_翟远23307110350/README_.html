Hénon map绘制程序

Hénon map的定义

数学上，Hénon map是一种位于离散时间上的动态系统。它是被广泛研究的代表混沌现象的动态系统之一。其迭代表达式为：
{xn+1=1−axn2+ynyn+1=bxn\begin{cases} x_{n+1}=1-ax_n^2+y_n\\ y_{n+1}=bx_n \end{cases}{xn+1​=1−axn2​+yn​yn+1​=bxn​​
该映射通过迭代产生轨迹。亦即，从一点 u0=(x0,y0)u_0=(x_0,y_0)u0​=(x0​,y0​)开始代入上述表达式得到下一点坐标，如此迭代得到轨迹[u0,u1,...,uN][u_0,u_1,...,u_N][u0​,u1​,...,uN​]

程序功能

本程序通过 python 编程实现以下功能：

  1. 计算 Hénon map 的函数：对任意输入aaa、bbb、初始值u0u_0u0​以及轨迹长度 NNN ，输出厄农映射的轨迹[u0,u1,...,uN][u_0,u_1,...,u_N][u0​,u1​,...,uN​]
  2. 利用编写的函数计算经典 Hénon map 的轨迹: 固定参数值为 a=1.4a=1.4a=1.4, b=0.3b=0.3b=0.3, u0=(0,0)u_0=(0, 0)u0​=(0,0) 时，输入NNN值，得到轨迹图。（以 x 为横坐标， y 为纵坐标）
  3. 利用编写的函数计算 Hénon map 的 orbit digram ，即固定 b=0.3b=0.3b=0.3 ，改变 aaa 后获得⼀系列 Hénon map 的轨迹，然后以 aaa 为横轴，xxx为纵轴绘制 orbit digram 图。
  4. 分析上述画出的orbit digram，找到 Hénon map 可以收敛到⼀条周期性轨道的 aaa 值，计算该 aaa 值对应的 Hénon map 的轨迹并绘图。

解答思路

  1. 基于迭代表达式利用循环结构计算 Hénon map 的函数
  2. 基于 1 的代码，利用 matplotlib 作出轨迹图
  3. 基于上述代码，利用嵌套循环结构作出不同 aaa 值对应 xxx 值的图
  4. 基于 3 的代码，增加 NNN 的大小作图，观察比较不同 aaa 中，不重合的 xxx 数据点数量的变换，其中数量不随 NNN 增加的即为可能的 aaa ；不断缩小 aaa 的范围，得到收敛到周期性轨迹的 a 值为0.8975.
